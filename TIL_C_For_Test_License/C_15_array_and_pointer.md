# 27강 배열과 포인터

- 예제(2012 계리직 컴퓨터 일반)

```C
#include<stdio.h>
int main(void){
    int a[4] = {10, 20, 30};
    int *p = a;
    p++;
    *p++ = 100;
    *++p = 200;
    printf("a[0]=%d, a[1]=%d, a[2]=%d\n", a[0], a[1], a[2]);
}
//내정답
10 20 100 출력 => 오답, ++위치 확인 잘할 것
```

- 임의의 공간 배정

| a    | a[0] | a[1] | a[2] | a[3] |
| ---- | ---- | ---- | ---- | ---- |
|      | 10   | 20   | 30   |      |
| 100  | 100  | 101  | 102  | 103  |
| p    |      |      |      |      |

- `p++` 수행
  - p = 100 => p = 101
- ` *p++ = 100` 수행
  - *p = 100 우선 수행 => 현재 p가 가르키는 공간 => 101 => a[1]
  - a[1] = 100
  - ++ 수행 => p = 102

| a    | a[0] | a[1]    | a[2] | a[3] |
| ---- | ---- | ------- | ---- | ---- |
|      | 10   | **100** | 30   |      |
| 100  | 100  | 101     | 102  | 103  |
| p    |      |         |      |      |

- `*++p = 200` 수행
  - ++우선 수행 => 현재 p가 가르키는 공간 => 103 
  - 103 => a[3] => *p = 200 수행
  - a[3] = 200

| a    | a[0] | a[1] | a[2] | a[3]    |
| ---- | ---- | ---- | ---- | ------- |
|      | 10   | 100  | 30   | **200** |
| 100  | 100  | 101  | 102  | 103     |
| p    |      |      |      |         |

- `printf("a[0]=%d, a[1]=%d, a[2]=%d\n", a[0], a[1], a[2]);` 수행
  - a[0]=10, a[1]=100, a[2]=30 출력

여기에 기록되는 모든 내용은 유튜브 흥달쌤 채널의 강의에서 개인 공부 목적으로 기록함
