# 43강 함수가 주소를 리턴

- 예제(2014 국회전산직 9급 컴퓨터 일반)

```c
#include<stdio.h>
int main(){
    int i;
    int x = 10;
    int *p;
    int a[100];
    for(i=0; i < 100; i++)
        a[i] = i*10;
    p = func(x, a);
    printf("sum=%d", x + a[0] + a[1] + p[0] + p[1]);
}

int *func(int a, int *x){
    a = a + 10;
    x= x+1;
    *x = *x*2;
    return x;
}
// sum = 70 출력
```

- for문까지 실행 했을 시

| i    | x    | p    |
| ---- | ---- | ---- |
|      | 10   |      |

| a    | a[0] | a[1] | a[2] | a[3] | a[...] |
| ---- | ---- | ---- | ---- | ---- | ------ |
| 100  | 0    | 10   | 20   | 30   | ...    |

- `p = func(x, a);`

  - `p = func(10, a);`

    ``` c
    int *func(int a, int *x){ 
    a = a + 10;
        // a = 10 + 10 => a = 20
        x= x+1;
        // x = 100(주소값임)+1 => 101
        *x = *x*2;
        // x의 값 = 101의 값 * 2 => 20
        return x;
    }
    ```

  - func() 는 주소값(포인터)를 리턴함 => 101을 리턴

  - `p = 101(주소값;`

- 최종결과

| i    | x       | p           |
| ---- | ------- | ----------- |
|      | 10(int) | 101(주소값) |

| a    | a[0] | a[1] | a[2] | a[3] | a[...] |
| ---- | ---- | ---- | ---- | ---- | ------ |
| 100  | 0    | 20   | 20   | 30   | ...    |

- `printf("sum=%d", x + a[0] + a[1] + p[0] + p[1]);`
  - x + a[0] + a[1] + p[0] + p[1] => 10 + 0 + 20 + p[0] + p[1]
    - p[0] = 101, p[1] = 102 => 20, 20
  - x + a[0] + a[1] + p[0] + p[1] => 10 + 0 + 20 + p[0] + p[1] => 10 + 0 + 20 + 20 + 20 => 70
- sum=70 출력
