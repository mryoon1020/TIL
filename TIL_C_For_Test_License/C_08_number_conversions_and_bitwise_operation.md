# 11강 진법변환, 비트연산

### 진법 변환

- **10진수 => 2진수**
  - 2로 계속 나누는 방법

|      |  19  |  C   |
| :--: | :--: | :--: |
|  2   |  9   |  1   |
|  2   |  4   |  1   |
|  2   |  2   |  0   |
|  2   |  1   |  0   |

- 19 => 10011	
  - 또다른 방법
  - 같은 원리, 좀더 빠른방법

|  32  |  16  |  6   |  4   |  2   |  1   |
| :--: | :--: | :--: | :--: | :--: | :--: |
|  0   |  1   |  0   |  0   |  1   |  1   |

- **2진수 => 8진수**

|  0   |  1   |  0   |  0   |  1   |  1   |
| :--: | :--: | :--: | :--: | :--: | :--: |
|  4   |  2   |  1   |  4   |  2   |  1   |
|  0   |  2   |  0   |  0   |  2   |  1   |

- 19 => 10011 => 23
  - 각자리별로 더한다
- 2진수 => 16진수

- 19 => 10011 => 13
